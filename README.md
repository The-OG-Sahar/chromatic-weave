# chromatic-weave
A generative art tool that weaves unique digital tapestries with algorithms.

‚ú® About

Chromatic Weave is an exploration into the beautiful intersection of code and creativity. I wanted to prove that algorithms could be more than just logic they could be a paintbrush.

With a single click, the app generates a completely unique, abstract piece of art. It uses randomized parameters for color, line thickness, and bezier curve paths to create a complex, layered "digital tapestry." No two pieces are ever the same. You can also download your creation as a high-quality PNG image.
 üöÄ Getting Started

Want to run it yourself? It's incredibly easy! There are no API keys or servers to set up.

Prerequisites
Just a modern web browser (like Chrome, Firefox, or Safari).

**Installation:**
1.  Clone this repository to your local machine.
   
    git clone https://github.com/the-og-sahar/chromatic-weave.git
    
2.  Navigate into the project directory.
    
    cd chromatic-weave
    
3.  Open the `art.html` file in your favorite web browser.

That's it! You can now start generating your own generative art.
